# TempConvert

A simple temperature converter website made by me for beest and it is totally cool ahahaha


## Features

- Convert Celsius into  and Fahrenheit into  Kelvin and kelvin to celsius
- also it automatically background changes background whenever the temperqaature gets cold, hot , normal
- the temperature conversion is very fast ,,,, just in one click
- special cool designs
- Reset button
- Simple and clean

## How To Use

- Enter temperature value wether its celsius or kelvin
- Select convert from unit
- Select convert to unit
- Click convert button
- View converted result also get a chance to see cool background

- HTML
- CSS
- Java
